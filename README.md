Bengaluru House Price Prediction (ML + Flask Web App)

A complete end-to-end Machine Learning project that predicts house prices in Bengaluru using XGBoost, advanced feature engineering, and a deployed Flask API + frontend.

This project demonstrates data science, ML engineering, API development, and deployment skills needed for real-world applications.

Project Features

- Cleaned and preprocessed raw housing data
- Feature engineering (BHK, bathrooms, outliers removal, scaling, location encoding)
- Trained XGBoost Regression Model
- Achieved a strong RMSE score
- Saved the trained model using Joblib
- Built a Flask backend API
- Created a HTML/JS frontend UI
- Predicts house prices with selected location, sqft, BHK, bathrooms
